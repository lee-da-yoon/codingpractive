1. 필드의 데이터 형식을 판단하는 단계

2. 작은 따옴표로 숫자와 텍스트를 구분하는 단계
    숫자일 경우
        생년=txt생년
    데이터 형식이 텍스트일 경우 
        학교='txt학교'

3. 큰 따옴표로 컨트롤을 구분하는 단계
    데이터 형식이 숫자일 경우
        "생년="txt생년
    데이터 형식이 텍스트일 경우
        "학교='"txt학교"'"

4. 앰퍼샌드(&)로 컨트롤과 연결하는 단계
    올바른 식을 완성하기 위해 컨트롤과 나머지 부분들을 앰퍼샌드로 연결해야 한다
    - 데이터 형식이 날짜일 때 조건식 작성 
        날짜일 경우 컨트롤에 붙이는 작은 따옴표 대신 #을 넣어서 구분하면 된다
        "수강일=#" &txt날짜& "#"
    - 컨트롤 값이 포함된 조건식 작성 
        데이터 형식이 텍스트이고 컨트롤 값이 포함된 데이터를 검색해야 한다면 컨트롤의 앞뒤로 *를 붙이고, = 대신 like 연산자를 사용한 후 텍스트 데이터 조건식 작성법을 따르면 된다. 
        "강좌명 like '*" &txt찾기& "*'"
        ' 아니 근데 왜 *가 '안에는 들어가면서 "안에도 들어가는거임??? 
        
    - 두 가지 조건을 모두 만족하는 조건식
        데이터 형식이 텍스트이고 동아리명, 코드명 두 가지 조건이 모두 일치하는 경우의 조건식을 세우려면 두 조건을 AND 연산자로 묶어 처리하면 된다. 
        "동아리명 = '" & txt동아리명 & "' AND 코드명 = '" & txt코드명 & "'" 
        ' 저기 앞에 다 & 붙여야되는건가
        ' 아 걍 앞뒤로 딴거 붙으니까 붙이는듯
